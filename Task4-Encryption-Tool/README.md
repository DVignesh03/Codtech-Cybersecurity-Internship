# Robust Encryption Application

### Description

This is a simple yet advanced encryption application. It uses AES-256 algorithm for the encryption and decryption. The application has a simple yet very much functional and user-friendly GUI for anyone to understand. All the user requires to enter is the password to encrypt the selected file.

### File structure
- **`main.py`**: The application script.
- **`requirements.txt`**: The file outlining the requirements.

### How To Run
- Navigate to the directory of the tool.
- Install the requirements.
- Run the application in Terminal using "python main.py".
- The GUI pops up. Enter the file's absolute path or select the file using the button below.
- Then enter a desired password to the encrypt the file for the first time. This password will be used to decypt the file as well.
- To decrypt the file select the encrypted file (which has the same name) or type ".enc" at the end of the path to decrypt.
